# IGRUS_projectContest
C++로 구현한 직관적이고 깔끔한 콘솔 기반의 할 일 관리 프로그램입니다.
사용자는 할 일을 추가, 수정, 삭제하고 완료 여부를 체크할 수 있으며, 작성된 목록은 텍스트 파일로 저장되어 프로그램을 껐다 켜도 데이터가 유지됩니다.

🚀 주요 기능 (Features)
할 일 CRUD 완벽 지원: 할 일 추가, 목록 확인, 수정, 삭제 기능을 제공합니다.

상세한 속성 관리: 각 할 일마다 제목, 기한, 카테고리, 우선순위(1~5)를 설정할 수 있습니다.

완료 여부 시각화: 완료된 항목은 [O], 미완료 항목은 [X]로 직관적으로 표시됩니다.

우선순위 정렬: 등록된 할 일들을 우선순위가 높은 순서대로 내림차순 정렬할 수 있습니다.

데이터 영구 저장: 할일.txt 파일에 데이터를 자동으로 저장하고 불러옵니다. (구분자 | 사용)

안정적인 입력 처리: 잘못된 입력으로 인한 무한 루프나 버퍼 꼬임 현상을 방지하는 예외 처리가 적용되어 있습니다.

🛠️ 개발 환경 (Environment)
Language: C++ (C++11 이상 권장)

OS: Windows 권장

Note: 화면 정리 및 대기열 처리를 위해 Windows 전용 명령어(system("cls"), system("pause > nul"))가 사용되었습니다. Mac/Linux 환경에서 실행하려면 해당 부분을 system("clear") 및 cin.get() 등으로 수정해야 합니다.

📋 메뉴 설명 (Menu Options)
프로그램을 실행하면 다음과 같은 메뉴를 제공합니다.

할일리스트확인: 현재 등록된 모든 할 일과 상태를 출력합니다.

할일추가하기: 새로운 할 일의 제목, 기한, 카테고리, 우선순위를 입력받아 추가합니다.

할일삭제하기: 목록 번호를 선택하여 해당 할 일을 완전히 삭제합니다.

할일수정하기: 목록 번호를 선택하여 해당 항목의 세부 정보를 다시 입력받아 수정합니다.

할일완료하기: 목록 번호를 선택하여 해당 항목의 상태를 [X]에서 [O]로 변경합니다.

우선순위정렬하기: 입력된 우선순위 숫자(5에 가까울수록 높음)를 기준으로 목록을 재정렬합니다.

파일저장하기: 현재까지의 변경 사항을 할일.txt에 덮어씌워 저장합니다.

종료: 1~7 이외의 숫자를 입력하면 프로그램이 안전하게 종료됩니다.

💾 파일 저장 구조 (Data Format)
데이터는 프로그램과 같은 폴더 내의 할일.txt에 저장되며, 각 항목은 | 기호로 구분됩니다.

Plaintext
제목|기한|카테고리|우선순위|완료여부(1/0)
(예시)
C++ 공부하기|2023-12-31|개발|5|0
▶️ 실행 방법 (How to Run)
코드를 복사하여 main.cpp 파일로 저장합니다.

C++ 컴파일러(g++, Visual Studio 등)를 사용하여 코드를 컴파일합니다.

생성된 실행 파일(.exe)을 실행합니다.

Tip: 만약 깃허브에 올리신다면 실행 화면을 캡처해서 README 상단이나 중간에 이미지로 하나 끼워 넣으시면 훨씬 보기 좋은 문서가 될 것입니다.
